# Проект по тестированию REST API (JSONPlaceholder)

## 📌 Описание проекта
Данный проект посвящен мануальному тестированию серверной части (API) учебного сервиса **JSONPlaceholder**. Сервис имитирует работу реального бэкенда интернет-магазина или блога (посты, пользователи, комментарии).

**Цель проекта:** Продемонстрировать навыки работы с REST методами, анализа JSON-ответов и проверки статус-кодов.

---

## 🛠 Инструменты и технологии
- **Postman:** Отправка запросов и валидация статус-кодов.
- **REST API Методы:** GET, POST, PUT, DELETE.
- **Формат данных:** JSON.
- **HTTP Статус-коды:** 200, 201, 404, 400.

---

## 🔍 Область тестирования (Scope)
Проверены следующие конечные точки (Endpoints):
- `/posts` — получение и создание постов.
- `/posts/1` — работа с конкретным ресурсом.

---

## 📂 Состав репозитория
1. `api-test-cases.md` — детальные сценарии проверок.
2. `api-bug-reports.md` — найденные несоответствия в работе API.
3. `postman_collection.json` — экспорт коллекции из Postman для импорта и проверки.
